# Scenario Template: new_script

## Goal
Create a script or CLI utility that automates a focused task safely and repeatably.

## Requirement Checklist
- Input/output definition
- Runtime environment and dependencies
- Failure and retry behavior
- Logging and observability expectations
- Idempotency requirements

## Deliverables
- Script implementation
- Usage examples (arguments/options)
- Unit tests for parsing and core behavior
- Integration test for representative execution path
- Progress update in `docs/progress/MASTER.md`

## Acceptance Criteria
- Script runs successfully with documented command
- Invalid inputs fail with clear error message
- Tests pass and coverage target is met
- Basic usage documentation is present
